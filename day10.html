Day 10: Final Project

Project Overview

Create a complete REST API for a Task Management System. This project will incorporate all the concepts learned throughout the course.

Project Requirements:

  • User authentication and authorization
  • Task CRUD operations
  • Task categorization and filtering
  • User profile management
  • API documentation
  • Testing and validation

API Endpoints

Authentication

POST /api/auth/register
POST /api/auth/login
POST /api/auth/logout
GET /api/auth/me

Tasks

GET /api/tasks
GET /api/tasks/:id
POST /api/tasks
PUT /api/tasks/:id
DELETE /api/tasks/:id
GET /api/tasks/category/:category
GET /api/tasks/status/:status

Users

GET /api/users
GET /api/users/:id
PUT /api/users/:id
DELETE /api/users/:id

Project Deliverables

  1. Complete API implementation with all endpoints
  2. Authentication system with JWT
